This custom hook is built upon [useBreakpoints()](https://design.wonderflow.ai/get-started/documentation/develop/hooks/useBreakpoints) and returns the provided configuration depending on the current breakpoint. It responds to window resizing and is built to work in server-side rendering (SSR) applications.

## Usage

```tsx lines="1,4,8,9,10"
import { useBreakpointsConfig } from "@wonderflow/react-components";

export default function Component() {
  const { value } = useBreakpointsConfig();

  return (
    <div>
      <p>{`Breakpoints: ${JSON.stringify(breakpoints)}`}</p>
      <p>{`Matches: ${matches)}`}</p>
      <p>{`Size: ${size}`}</p>
    </div>)
}
```
